   - 新增 `/health`，返回 JSON：`{"status":"ok","service":"doh-proxy","endpoint":"/dns-query"}`。
   - 新增 `jsonHeaders()`。
   - POST 转发从 `body: request.body` 改为 `body: await request.arrayBuffer()`，提升本地/Bun/Cloudflare 兼容性。
2. `functions/health.js`
   - 新增 Pages Functions 版 `/health`。
3. `functions/dns-query.js`
   - POST 转发同步改为 `body: await request.arrayBuffer()`。
4. `scripts/test-doh.js`
   - 新增 `/health` 测试。
   - 新增 bad POST=415、bad path=404 边界测试。
   - 修复测试判定：预期 404/415 不再算失败。
   - 将旧空访问 `/dns-query` 检查命名为 `dns-query`，避免和 `/health` 混淆。
5. `README.md`
   - 补充 `/health` 说明。
